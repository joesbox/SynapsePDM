# JuicePDM
### CAN Enabled Automotive Power Distribution Module

#### Features:
+ 6 High Side Driver output channels.
+ Up to 13A output per channel.
+ Configurable PWM (200Hz) or digital outputs.
+ Configurable load current and channel diagnostics.
+ 6 configurable digital inputs.
+ CAN control and diagnostics.
+ SD card logging.
+ Configuration and monitoring via USB.
+ Battery-backed RTC.
+ Expansion via SPI or I<sup>2</sup>C.
